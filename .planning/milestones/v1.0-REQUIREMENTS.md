# Requirements

## v1 Requirements

### Core Stability
- [x] **CORE-01**: Refactor core simulation engine to eliminate 136 bare exceptions, consolidate the 15+ versions, and ensure errors crash loudly rather than failing silently.
- [ ] **CORE-02**: Map thermodynamic conditions and improve solver logic to reach a ≥98% convergence rate across standard time steps.
- [ ] **CORE-03**: Improve control logic beyond simple temperature limits to maximize solar fraction and minimize auxiliary heat demand.

### Architecture & Physics
- [ ] **ARCH-01**: Support the existing Parallel configuration cleanly.
- [ ] **ARCH-02**: Implement the new Series integration configuration.
- [ ] **ARCH-03**: Replace air with Molten Salt as the single HTF inside the PBTES, bypassing/removing the heat exchangers in the simulation.

### Metrics & Analysis
- [ ] **METRICS-01**: Dynamically calculate pump power. Attempt to include pressure as a variable in the Molten Salt simulation; if off-design convergence fails, fallback to post-processing pump power based on mass flow.
- [ ] **ANALYSIS-01**: Develop an automated parametric script for sweeping variables (temperatures, control logic, volume, particle size) and outputting consolidated results.
- [ ] **ANALYSIS-02**: Produce publication-quality (Q1 scientific article) graphs for both single simulations and parametric sweeps.
- [ ] **ANALYSIS-03**: Synthesize initial and final results to assess and formulate the narrative of the scientific article.
- [ ] **ECON-01**: Implement the Levelized Cost of Heat (LCOH) economic model based on CAPEX/OPEX sizing.
